# Strategy Templates by Site Type

Templates for different business types to guide the page architect and strategy writer.

## Content Site (Blog/Magazine)

### Goal
Maximize organic traffic for ad revenue or affiliate income.

### Typical Metrics
- 200-500 pages
- 80% informational content
- Focus on high-volume topics
- Build topical authority clusters

### Phase Structure

**Phase 1: Foundation (Month 1-2)**
- Create 8-12 pillar pages for main topics
- Target mid-volume, lower-difficulty keywords
- Establish content categories

**Phase 2: Cluster Expansion (Month 3-4)**
- Build 40-60 cluster pages
- Target long-tail variations
- Internal link to pillars

**Phase 3: Long-tail Capture (Month 5-6)**
- Create 100+ supporting articles
- Target question-based keywords
- Update and interlink existing content

### Priority Multipliers
| Content Type | Multiplier |
|--------------|------------|
| Listicle (traffic) | 1.5x |
| How-to guide | 1.3x |
| Comparison | 1.2x |
| Glossary | 0.8x |

## SaaS / Software

### Goal
Generate qualified leads and trials.

### Typical Metrics
- 80-150 pages
- 60% commercial/transactional content
- Feature and solution pages
- Comparison and alternative content

### Phase Structure

**Phase 1: Convert (Month 1-2)**
- Optimize existing product/feature pages
- Create comparison pages (vs competitors)
- Create pricing/alternatives pages
- Target BOFU keywords

**Phase 2: Capture (Month 3-4)**
- Create solution pages by use case
- Create integration pages
- Target MOFU keywords
- Build case studies

**Phase 3: Educate (Month 5-6)**
- Create educational blog content
- Target TOFU keywords
- Build email capture funnels
- Establish thought leadership

### Priority Multipliers
| Content Type | Multiplier |
|--------------|------------|
| Comparison (vs X) | 2.5x |
| Alternatives | 2.0x |
| Feature page | 1.8x |
| Integration | 1.5x |
| How-to guide | 1.2x |
| Educational | 0.8x |

## Local Services (Plumber, Lawyer, etc.)

### Goal
Generate local leads and phone calls.

### Typical Metrics
- 30-80 pages
- Location pages for service areas
- Service-specific pages
- Cost/pricing guides

### Phase Structure

**Phase 1: Core Services (Month 1)**
- Create/optimize main service pages
- Add cost guide pages
- Target "[service] [city]" keywords

**Phase 2: Location Coverage (Month 2-3)**
- Create location template pages
- Cover all service areas
- Local schema markup

**Phase 3: Educational Support (Month 4+)**
- Create helpful guides
- FAQ pages
- Emergency guides (when applicable)

### Priority Multipliers
| Content Type | Multiplier |
|--------------|------------|
| Service + location | 3.0x |
| Cost guide | 2.0x |
| Service page | 1.8x |
| FAQ | 1.2x |
| Educational | 0.6x |

### Location Template Strategy
- One template page, not 50 city pages
- Dynamic content for each location
- Target: "[service] [city]" pattern

## Template Usage Notes

1. **Adapt, don't copy:** These are starting points, not rigid rules
2. **Consider resources:** Smaller teams need smaller scope
3. **Phase timing:** Adjust based on content velocity
4. **Multipliers are guides:** Use judgment for edge cases
5. **Business goals trump templates:** If the client wants traffic over leads, adjust accordingly
